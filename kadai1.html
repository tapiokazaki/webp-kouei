日本大学文理学部情報科学科Webプログラミングの演習課題

5421018 岡崎 光栄

<body>について

  1. 説明
    2. bodyは文章や画像等をブラウザ画面上に表示される内容を指定するタグである。
  2. マークアップ例
    3. <body> '文字など' </body>
  3. 動作結果
    4. '文字など'がwebブラウザー画面に出力された。

<h1>について

  1. 説明
    2. webブラウザー上の一番最初の見出しを出力するタグ。
  2. マークアップ例
    3. <h1> '文字など' </h1>
  3. 動作結果
    4. '文字など'が太文字でwebブラウザー画面一番上に出力された。

<p>について

  1. 説明
    2. 段落を作るためのタグ。
  2. マークアップ例
    3. <p>'文字など'<p>
  3. 動作結果
    4. 次の文章と'文字など'が段落が分けられて出力された。
